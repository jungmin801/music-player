## 1. 프로젝트 소개
- 오디오 파일을 업로드하여 웹에서 음악을 재생할 수 있는 웹 플레이어입니다.
- Firebase Storage와 Firestore를 이용하여 재생 목록을 저장하고, 저장한 목록을 불러올 수 있습니다.

## 2. 실행 방법
- 좌측 하단의 파일 추가 버튼을 클릭하여 원하는 mp3 혹은 wav 파일을 업로드하세요.

## 3. Stacks
- React
- Javscript
- Styled-Components
- Recoil
- HTML5
- Firbase
- Figma
- jsmediatags : 음악 파일에서 메타데이터를 추출하기 위해 사용
